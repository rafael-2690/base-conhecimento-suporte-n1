# Erro de Login no Windows

## Objetivo

Resolver problemas relacionados ao login de usuários no Windows.

## Passos

1. Verificar se o usuário está digitando a senha correta.
2. Reiniciar o computador em modo de segurança, se necessário.
3. Verificar conectividade com o controlador de domínio (para contas de domínio).
4. Redefinir a senha ou desbloquear o usuário no Active Directory.
5. Checar logs de evento para erros de autenticação.

## Observações

- Confirmar se houve alterações recentes no perfil ou políticas de senha.
