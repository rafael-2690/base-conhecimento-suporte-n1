# Formatação do Windows

## Objetivo

Descrever o procedimento padrão para formatação e reinstalação do Windows em máquinas corporativas.

## Pré-requisitos

- Backup dos dados do usuário
- Mídia de instalação ou imagem do sistema
- Licença válida do Windows

## Passos

1. Confirmar necessidade da formatação.
2. Realizar backup completo dos dados.
3. Preparar mídia ou imagem de instalação.
4. Executar instalação limpa do Windows.
5. Instalar drivers e softwares necessários.
6. Restaurar dados do backup.

## Observações

- Verificar se todos os drivers estão atualizados.
