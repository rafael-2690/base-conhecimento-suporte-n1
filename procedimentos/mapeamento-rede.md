# Mapeamento de Rede

## Objetivo

Explicar como mapear unidades de rede e recursos compartilhados para usuários.

## Pré-requisitos

- Caminho da pasta de rede
- Permissões de acesso ao recurso

## Passos

1. Abrir o Explorador de Arquivos.
2. Selecionar "Mapear unidade de rede".
3. Inserir o caminho do recurso.
4. Escolher letra de unidade e conectar usando credenciais se necessário.

## Observações

- Confirmar permissão de acesso antes de mapear recursos.
